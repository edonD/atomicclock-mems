## Method Notes

Minimum trace width is determined per polygon using the **bounding-box
minimum dimension** (min of width and height). For the axis-aligned
rectilinear traces and turn segments used in this layout this equals the
exact trace width. Bond pad squares are also on these layers; their
150 × 150 µm size passes the width rule trivially.
